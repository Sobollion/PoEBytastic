# PoEbytastic

Мештастик нода, основанная на модулях e22-900m30s/e22-400m30s/e22-900m33s/e22-400m33s и девборде Waveshare-ETH (микроконтроллер ESP32 s3) с питанием от PoE модуля от Waveshare.
* ВНИМАНИЕ! Не запитывайте устройство одновременно от PoE и USB!

# Описание платы

<img src="./RU description.png" width="400">
* Плата сделана из faketec v4, не имеет отдельного внешнего питания, антенный разъём SMA F, опциональные подтяжки для i2c датчиков и мосфеты для опциональной периферии.
